# ETH Price Tracker

This project was created as part of a technical assignment for a Customer Solution Engineer position.

A simple Java application that retrieves the ETH/USD price using the Tatum API and tracks price changes over time.

---

## Features

- Retrieves cryptocurrency price from Tatum API
- Parses JSON response
- Displays formatted output
- Calculates price difference between two measurements

---

## Requirements

- Java
- Maven
- Tatum API key

---

## How to run

### 1. Set API key (PowerShell)

```
$env:TATUM_API_KEY="your_api_key"
```

### 2. Run the application
